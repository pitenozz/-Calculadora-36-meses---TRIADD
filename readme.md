# Calculadora 36 meses - TRIADD

Pequena aplicação web para somar períodos de trabalho e verificar se o total atinge 36 meses (CLT).

## Como usar localmente
1. Salve `index.html` e `style.css` na mesma pasta.
2. Abra `index.html` no navegador (duplo clique).

## Publicar no GitHub Pages
1. Crie um repositório público no GitHub.
2. Faça upload dos arquivos (`index.html`, `style.css`, `README.md`).
3. Em Settings → Pages, selecione a branch `main` e a pasta `/root`.
4. Aguarde alguns minutos e acesse `https://seu-usuario.github.io/nome-do-repositorio/`.

## Observações
- O cálculo usa aproximação de 1 mês = 30.44 dias para somatório.
- Para qualquer ajuste visual ou funcional, edite `index.html` e `style.css`.
